## ⚡ 快速操作路径

### 每日必做（5分钟）

#### 1. 快速抓取（2分钟）
```
登录百度站长 → 左侧菜单"资源提交" → "快速抓取"
→ 输入 URL → 提交
```

**提交内容**：
```
https://www.sswl.top/
https://www.sswl.top/posts
```

---

#### 2. 查看索引量（1分钟）
```
左侧菜单"数据统计" → "索引量" → 查看曲线图
```

**查看要点**：
- 今天的索引量是多少？
- 和昨天相比是否增长？

---

#### 3. 查看抓取频次（1分钟）
```
左侧菜单"数据统计" → "抓取频次" → 查看图表
```

**查看要点**：
- 今天百度来了多少次？
- 频次是否稳定？

---

### 每周任务（10分钟）

#### 周一：查看搜索流量
```
左侧菜单"数据统计" → "流量与关键词"
```

**查看内容**：
- 展现量、点击量
- 关键词排名
- 热门页面

---

#### 周三：手动提交分类页
```
左侧菜单"资源提交" → "普通收录" → "手动提交"标签
→ 输入分类页 URL → 提交
```

**提交内容**：
```
https://www.sswl.top/categories
https://www.sswl.top/categories/movies
https://www.sswl.top/categories/books
https://www.sswl.top/categories/tv-series
https://www.sswl.top/categories/documentaries
https://www.sswl.top/categories/software
https://www.sswl.top/categories/music
https://www.sswl.top/categories/education
https://www.sswl.top/categories/others
https://www.sswl.top/about
```

---

#### 周五：抓取诊断
```
左侧菜单"数据统计" → "抓取诊断"
→ 输入 URL → 点击"诊断"按钮
```

**测试页面**：
```
https://www.sswl.top/
https://www.sswl.top/posts
```

**查看结果**：
- ✅ HTTP 200：正常
- ❌ 其他状态码：需要检查

---

#### 周五：检查抓取异常
```
左侧菜单"数据统计" → "抓取异常"
```

**查看要点**：
- 是否有异常 URL？
- 异常原因是什么？
- 需要修复吗？

## 💡 操作技巧

### 技巧1：批量提交 URL
在"快速抓取"或"手动提交"时：
- 一行一个 URL
- 不要有多余空格
- 确保 URL 完整（包含 https://）

### 技巧2：优先级排序
配额有限，优先提交：
1. 首页 > 列表页 > 详情页
2. 新文章 > 旧文章
3. 重要页面 > 普通页面

### 技巧3：定时操作
- 每天固定时间操作（如早上9点）
- 形成习惯，不会遗漏
- 便于观察规律

### 技巧4：记录数据
建议记录每天的：
- 索引量
- 抓取频次
- 推送数量

用于分析趋势和效果
