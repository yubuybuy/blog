## 📝 使用步骤(只需6步)

### 1. 访问页面
```
http://localhost:3000/wechat-publisher
```

### 2. 选择文章
- 左侧列表点击你要发布的文章
- 默认选中第一篇(最新的)

### 3. 复制内容
- 点击 **"📋 一键复制文章内容"** 按钮
- 看到 ✅ **"内容已复制到剪贴板"** 提示

### 4. 打开公众号后台
- 访问: https://mp.weixin.qq.com/
- 点击 **素材管理 → 新建图文素材**

### 5. 粘贴内容
- 在编辑器中按 **Ctrl+V**
- 内容自动粘贴,格式完整

### 6. 设置链接和发布
- 复制"阅读原文链接",设置到公众号
- 如果需要,复制封面图链接并下载设置
- 预览检查,发布!

**总耗时: 约30秒!**

## 🔧 技术实现

### 创建的文件

```
src/app/
├── api/
│   └── wechat-content/
│       └── route.ts              # API: 获取并转换文章
├── wechat-publisher/
│   └── page.tsx                  # 公众号生成器页面
└── boss-admin/
    └── page.tsx                  # 后台(已添加公众号标签)
```

### API接口

**GET /api/wechat-content?limit=20**

返回格式化的公众号文章列表,包含:
- 标题
- 作者
- 摘要(已添加导流)
- 内容(已格式化+导流+免责声明)
- 文章链接
- 封面图链接
