## 🎯 快速开始

### 第一步: 运行命令

```bash
# 生成1篇文章(推荐)
npm run wechat:personal

# 批量生成3篇文章
npm run wechat:personal:batch
```

### 第二步: 浏览器自动打开

工具会:
- ✅ 自动生成格式化的内容
- ✅ 自动在浏览器中打开预览页面
- ✅ 显示一键复制按钮

### 第三步: 一键复制

在打开的页面中:
1. 点击 **"一键复制内容"** 按钮
2. 看到 ✅ **"内容已复制到剪贴板"** 提示

### 第四步: 粘贴发布

1. 登录公众号后台: https://mp.weixin.qq.com/
2. 点击 **素材管理 → 新建图文素材**
3. **Ctrl+V** 粘贴内容
4. 上传封面图(工具会显示图片链接)
5. 点击 **保存并群发**

**总耗时: 约15秒!**

## 🚨 注意事项

### 关于"阅读原文"链接

**重要**: 粘贴内容到公众号后,记得设置"阅读原文"链接!

在公众号编辑页面:
1. 找到 **"原文链接"** 设置
2. 填入你的文章链接,例如: `https://www.sswl.top/posts/article-slug`
3. 工具生成的HTML中会显示正确的链接

### 关于封面图

工具会显示文章的封面图链接,你需要:
1. 下载该图片
2. 或在公众号后台重新上传
3. 设置为文章封面

### 关于内容审核

确保内容符合公众号规范:
- ✅ 已按照 `AI_CONTENT_RULES.md` 优化
- ✅ 避免版权敏感词汇
- ✅ 包含免责声明
